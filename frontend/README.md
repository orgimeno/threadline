# Threadline frontend

This workspace contains the Vue 3, Vite, and TypeScript frontend for Threadline.

The current implementation is an application shell. It supports local selection of JSON and Markdown files and presents the planned import, review, and export workflow. It does not upload files, call OpenAI, mutate entries, or create real exports yet.

Run commands from the repository root whenever possible. See the root [README](../README.md) for setup, development, and verification instructions.
