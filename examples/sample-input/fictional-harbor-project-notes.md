# Harbor workshop — fictional project notes

**Project:** Harbor is a fictional online workshop for local artists to publish short process notes and exchange peer feedback.

## Decisions from the planning session

- The project name is **Harbor**. An earlier working title, **Canvas**, is retired.
- Weekly project updates are written on Friday mornings and should contain at most five concise bullet points.
- The onboarding session is booked for **May 6, 2031, 10:00, Europe/Madrid**.
- Shared summaries must exclude personal addresses, contact details, and private messages.

## Upcoming work

- The first prototype review is scheduled for **May 20, 2031**.
- During that review, a participant should be able to publish one process note and request peer feedback without assistance.
- The first-release success target is five fictional artists completing onboarding and each leaving one useful peer comment.

## Open question

The team has not yet decided whether peer feedback should be visible to every participant by default or only to the original author and invited reviewers.
