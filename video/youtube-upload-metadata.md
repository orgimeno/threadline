## Description

```text
Threadline turns scattered AI conversation exports into structured, traceable context that people review before exporting.

In this demo, I import JSON and Markdown notes, process valid sources with GPT-5.6 through the OpenAI Responses API, review each proposed entry, and export the final context as JSON and Markdown.

Threadline keeps people in control: AI proposes structured context, while the reviewer decides what to accept, edit, reject, or reopen. Every proposal keeps a reference to its original file and location.

Built for OpenAI Build Week with Codex and GPT-5.6.

Codex helped define the product scope and canonical schema, make architecture decisions, build the Vue frontend and Fastify backend, write tests and documentation, and debug the live extraction flow.

GPT-5.6 handles the unstructured interpretation and extraction step. Threadline validates the output, preserves provenance, and makes the final decision a human one.

Repository: https://github.com/orgimeno/threadline

The repository includes setup instructions, fictional sample sources, and a no-key Demo Mode for a repeatable review and export walkthrough.

Narration generated with OpenAI text-to-speech.
```
